This is a simple text file transfer application.

The client is allowed to upload and download text files

to and from the server. The application is straight forward.

		Upload or Download Menu
	Select upload if you would like to upload a text file to the server.
	Select download if you would like to download a text file to the server.

		If upload was selected you can select a text file from the upload file
		menu and press the upload button to upload the file.
		There will be a window that pops up where you can rename your 
		file before you send it if you choose to. Select save and your file	
		will be uploaded.

		If the download button was selected choose a text file from the menu
		that you wish to download and select the download button.
		Your text file will be downloaded to your directory.
